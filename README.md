# Chrome Extension to Add Subtitles

Enhance Your Streaming Experience with Custom Subtitles! With this extension, you can take full control of your video subtitles on Netflix, Prime Video, YouTube, and more. text customizable options, settings to decide the position of the captions, and a timing sync tool are included. A user-friendly interface will make it easier for you to set up everything.

[Install from Chrome Web Store](https://chromewebstore.google.com/detail/substream-add-subtitles-t/miebboahepoojbiboedchfbdemcglnnh)

## Key Features

- Load subtitles into online streaming platforms such as Netflix, Prime Video, and YouTube.
- Adjust font style, size, shadow, background color, and caption alignments.
- Adjust subtitle timing using the easy synchronization tool.
